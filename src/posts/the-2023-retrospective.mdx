## Retrospective 2022

### Resignation

Batch Dua Kampus Merdeka telah dibuka pendaftarannya untuk mahasiswa semester
genap. Aku berpikir bahwa itu merupakan sebuah kesempatan bagiku. Namun sayang,
program tersebut memerlukan dedikasi waktu _full-time_, dan memerlukan
pernyataan tidak sedang bekerja. Sehingga, dari tempat lama aku bekerja sebagai
full-time software engineer, aku memutuskan resign.

### Welcome, New Stack

Dengan ikutnya aku dalam program tersebut, bertambahlah stack bahasaku.
Awalnya, aku hanya masih fokus dengan bahasa skrip, seperti PHP atau sebangsa
NodeJS. Namun sekarang, dengan masuknya Golang dalam kurikulum, aku makin
condong dengan bahasa terkompilasi. Selain itu, pengenalanku terhadap desain
back-end dan database makin dalam.

### Project Freelance

Setelah resign, aku mencoba untuk menerima beberapa permintaan project kecil.
Project-project tersebut berbentuk project aplikasi web berbasis PHP dan
sebagian berbasis ReactJS.

### Here We Go, Jobseeker!

Setelah menyelesaikan program Kampus Merdeka, aku beralih fokus untuk mencari
kerja lagi sebagai full-stack engineer. Sekarang, aku memilih cabang yang lebih
luas, yakni dalam bidang multiplatform (mobile) developer dengan Flutter dan
React Native, selain masih fokus dengan web development.

### Kerja, Kerja, Kerja!

Setelah sekitar tiga bulan, aku mendapatkan pekerjaan sampai saat artikel ini
ditulis dan dipublikasikan, sebagai full-time web developer. Walaupun kerja
hybrid beda kota, tetapi okelah.
