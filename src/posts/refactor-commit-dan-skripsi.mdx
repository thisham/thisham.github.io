Akhirnya gua bisa ngeluangin waktu buat nge-refaktor web ini! Hampir aja repo 
ini gua kategorikan sebagai "abandoned project" lagi kalau gua bener-bener gak 
ngeluangin waktu. Padahal, gua bikinnya cukup effort buat melajarin parsing 
**mdx** ke react element. Ya, mungkin tech decission-nya cukup sulit kali,
makanya effort-nya kerasa banget.

Gua refaktor semua lini web ini jadi pake tailwind. Jujur, gua sekarang lebih 
nyaman pake tailwind daripada component-component yang udah jadi. Gua serasa 
lebih berkuasa buat customize tampilannya jadi kaya yang gua inginkan. Hasil 
refaktor dan redesign gua attach di gambar 1 dan gambar 2. Desain ini gua buat 
fully inspired by [enji.dev](https://enji.dev), tapi dengan banyak modifikasi.

Awalnya, rencana gua di web ini hanya buat post blog aja. Tapi makin ke sini 
gua malah berekspektasi buat nambahin section portofolio di sini. Semua gara-
gara tailwind sama kerjaan! No debat! Gua pindah alur jadi frontend saat ini, 
padahal sertifikasi banyakan dari backend. Heran gua, ngelamar fullstack 
dapetnya bagian frontend mulu.

Maintenance web ini mungkin bakal cukup teratur aja kali, ya. Ekspektasi gua 
kali ini, web ini bakal gua gunain buat show-off beberapa project yang udah 
pernah gua kerjain. Baik itu project dari platform web maupun (rencananya) 
mobile juga. Sayang banget akun GPlay Console gua anggurin, ada kali setahun 
lebih. Mostly, project yang bakal gua show-off di web ini ya project freelance 
sama built for fun doang, itupun tampilan atau mock-up aja (dan kalau boleh). 
Ga mau gua berurusan sama hak cipta.

Selain itu, gua lagi bikin skripsi sih ini. Mungkin bakal banyak post setelah 
ini. Gua ga terlalu peduli bakal ada yang baca apa ngga, yang penting gua ga 
perlu cari berulang-ulang dengan topik yang sama. Gua akuin, skripsi gua susah 
bat, referensi-nya susah bener carinya. Kalau ada pun, usianya udah 5 tahun ke 
atas atau malah sangat baru. Gua ngincer yang baru juga, tapi yang pasti harga 
_paper_ per bijiannya itu mahal. Ada yang nyampe USD 35, belum ketambahan VAT 
tuh. Dilema anak skripsi ada-ada aja dah. 

Intinya gua seneng lah ya, bisa nerusin project kecil ini. Gua jadi ada tempat 
buat nge-post beberapa bahan skripsi sebelum gua masukin ke buku utama. Gua 
juga ingin gua bakal bisa teratur ngerawat project ini biar bisa gua pake 
sebagai portofolio gua ke depannya.
